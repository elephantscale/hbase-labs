Interacting with HBase in Python
----

Using HappyBase : http://happybase.readthedocs.org/en/latest/
install happybase by
  $   sudo pip install happybase

Start thrift server in HBase
  option 1)    bin/hbase-daemon.sh   start thrift


Edit test.py
update hbase host pointers

Execute python script
  $   python   test.py
